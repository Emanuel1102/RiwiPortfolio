Bienvenido al repositorio de mi portafolio personal. Este proyecto ha sido diseñado para mostrar mi perfil profesional, habilidades técnicas y los proyectos en los que he trabajado como desarrollador.

Mi enfoque actual es el desarrollo **Front-End**, sin embargo estoy abierto a cualquier oportubidad, ya que me apasiona el desarrollo de software en general y aprender constantemente.

## 🛠️ Tecnologías Utilizadas
Este proyecto fue construido utilizando tecnologías web estándar para garantizar rendimiento y compatibilidad:

*   **HTML5:** Estructura semántica avanzada.
*   **CSS3:** Estilos modulares, CSS Grid, Flexbox y variables personalizadas (`:root`).
*   **JavaScript:** Lógica de interacción para menús responsivos.
*   **Metodología BEM:** Utilizada para un nombrado de clases claro y escalable.

## 📦 Estructura del Proyecto
El proyecto está organizado de forma modular para facilitar el mantenimiento:

*   `index.html`: Archivo principal con las secciones de Hero, Sobre mí y Proyectos.
*   `styles/`: Directorio que contiene archivos CSS segregados por componentes (header, nav, main, footer).
*   `public/`: Carpeta de recursos para imágenes, iconos y otros activos estáticos.
*   `src/scripts/`: Lógica de JavaScript para el funcionamiento del sitio.

## 🌟 Características Principales
- **Diseño Responsivo:** Adaptado para dispositivos móviles, tablets y escritorio.
- **Navegación Fluida:** Implementación de `scroll-behavior: smooth` y ajustes de `scroll-margin-top` para una mejor experiencia de usuario.
- **Interactividad:** Menú tipo hamburguesa funcional en dispositivos móviles.
- **Efectos Visuales:** Uso de `backdrop-filter` para efectos de desenfoque y transformaciones CSS en tarjetas de proyectos.
